Simplifying Chord Recognition: A Case Study with NEK

This case study explores NEK, an open-source MIDI and audio chord recognition plugin, designed to simplify chord detection in educational applications. By analyzing both MIDI and audio inputs, NEK provides an accessible framework for developers aiming to build chord recognition tools. Download the paper below to learn more about the chord recognition algorithms NEK uses and how you can incorporate them. You can also access the NEK plugin to delve into the code or play around with it.

Chord Recognition Approach

To reiterate, NEK is a music education software plugin that functions as a MIDI or audio chord recognition tool. For audio, NEK utilizes an algorithm by Adam Stark, which breaks down an audio signal into frames and assigns a label based on the energy in the chromagram.

In the case of MIDI events, it projects the state of MIDI messages to a buffer and uses an intuitive approach to name the intended chord irrespective of the chord inversion.
